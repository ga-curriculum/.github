# GA Curriculum | Org Public Landing Page (source)

This repository holds the **public landing page** for the `ga-curriculum` organization — the page that non-members, students, and anyone on the internet see when they visit the org.

## Editing the page
The content that renders on the org profile lives in **`profile/README.md`**. Edit that file to change the landing page (use the pencil icon to edit it in the browser). Changes to this root `README.md` do **not** appear on the profile.

> **This repository is public.** Everything committed here — including this file — is visible to anyone on the internet and indexed by search engines. Keep it free of client names, internal links, and anything sensitive.

## Contents
- `profile/README.md` — the public landing page
- `assets/` — images used by the page

The member-only landing page (what signed-in org members see) is maintained separately in the `.github-private` repository.
